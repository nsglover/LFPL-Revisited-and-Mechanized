\section{Soundness Proof Cases}
\label{app:soundness}

Here, we provide some interesting cases of the soundness proof. First, the function application case, demonstrates the method by which we are able to simultaneously prove termination and a polynomial cost bound. Second, the list recursor case, is an exemplary case displaying usage all of the tools we developed to approach the proof. For convenience, we restate the theorem:
\begin{thm}[General Polynomial-Time Soundness]
For every $k \in \mathbb N$, suppose that $\types{\ctx}{M}{A}$ and $\env : \ctx$. If $n \ge \envsize{\env}$ and $k \ge P_M(n) + P_{\env}(n)$, then there is a value $v : A$ and a cost $c \in \mathbb N$ such that $\evals{\env}{M}{c}{v}$ and:
$$c + P_v(n) \le P_M(n) + P_{\env}(n)$$
\end{thm}
\begin{proof}
By strong induction, we may assume the theorem holds for all $k' < k$. Now we go by rule induction on $\types{\ctx}{M}{A}$.

For the application case, we assume that $\types{\ctx}{M N}{B}$ because $\types{\ctx}{M}{\tparr{A}{B}}$ and $\types{\ctx}{N}{B}$.
\end{proof}
